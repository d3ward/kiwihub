Set Up a New Tab Page

Customizing your new tab page (NTP) can significantly enhance your browsing experience by improving productivity and providing quick access to frequently visited sites
This guide will focus on setting up my "Next NTP" as an example.
However, you are encouraged to explore other NTP solutions/extensions that you might prefer, which can be found online or through the Chrome Web Store.

Option 1: Online Version

If you prefer not to install an extension, the online version is a quick and easy alternative.

  • Launch your browser (this guide assumes you're using Chrome or a Chrome-based browser like Kiwi).
  • Navigate to the settings menu, often found in the toolbar corner as three dots or lines.
  • In the settings menu, look for a section labeled "On startup" or "Homepage."
  • Select the option that allows you to set a specific page as your homepage.
  • Enter the URL: https://d3ward.github.io/nextntp and save your changes.
Every time you open your browser or click the home button, it will now take you to the Next NTP page.

Option 2: Extension Version

For those who prefer using an extension, follow these steps to install Next NTP via its extension version. This guide uses Kiwi Browser as an example, a popular choice for extension support on Android OS

Visit the Next NTP releases page on GitHub:
NextNTP Releases
Download the latest version of the extension. It may come in a .crx file or a .zip file.

Install the Extension in Kiwi Browser:
  • Open Kiwi Browser on your device.
  • Type chrome://extensions in the address bar and hit enter to open the extensions page
  • Enable "Developer mode" at the top of the page
  • If you downloaded a .zip file, extract it on your device. If it's a .crx file, keep it as is.
  • Tap on “Load unpacked” if you have an extracted folder, or "Install from file" if you have the .crx file, and select the downloaded file or extracted folder
  • After installing the extension, type chrome://newtab in the address bar and hit enter. This should now open the Next NTP page
  • To make it your default new tab page, go to the browser settings, find the "On startup" or "Homepage" section, and set chrome://newtab as your homepage
